# 擎添工業有限公司官方網站

擎添工業有限公司（Ching-Tech Industrial Co., Ltd.）成立於 1984 年，專注於工業自動化、智慧工廠規劃與建置，擁有超過 40 年的豐富產業經驗。

## 技術棧
- 純 HTML5 / CSS3 / JavaScript
- GitHub Pages 靜態部署
